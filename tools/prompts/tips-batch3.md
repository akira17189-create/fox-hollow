# 任务：为《狐狸谷》34 座建筑各写一句 tip

《狐狸谷》是静谧氛围的中式叙事增量游戏，主角是住山谷里的小狐狸。
这批 34 座建筑分两类：**宗教 / 神性**（27 座）和 **外交**（7 座）。

## 写作风格

- 长度 **7-13 个汉字**（中间逗号算 1 字），**绝不超过 13**
- 基调：**肃穆为底，但要藏一点人味、俏皮、反差、荒诞、或一个具体的小动作**
- 不要纯抽象白描——上一批 35 条里大量"太安静"的版本被驳回
- 要有：**具体的物件 / 具体的声响 / 具体的动作 / 狐狸的反应 / 一点点出乎意料**
- 主语可以是：建筑本身 / 它（指狐狸）/ 信徒、来客 / 物件
- **禁词**：突然 / 忽然 / 能量 / 力量 / 神奇 / 无比 / 感慨 / 欣慰 / 惊喜 / 奇迹 / 沉默下来 / 安静下来 / 沉默了一会儿
- **禁套式**：`不是X，是Y` / `不是X而是Y` / `是X，不是Y` / `X，是X`
- **禁机制术语**：解锁 / 激活 / 系统 / 分支 / 阶段 / 版本 / 配方 / 产出 / 效率 / 上限 / 容量
- **禁复述 desc**：句子不能跟下面给的描述近似
- 宗教题材**不要写得太二次元**，但可以有黑色幽默、荒诞细节、信徒的小动作
