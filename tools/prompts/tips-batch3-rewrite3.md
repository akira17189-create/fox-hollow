# 任务：为《狐狸谷》15 座建筑各重写一句 tip（电波 / 无厘头，**句式必须多样**）

电波方向对了，但上一轮 23 条里大量用「它做X，物体：说话」这个冒号对话句式，**被驳回——太重复**。

## 用户刚通过的 8 条（**研究它们的句式，每条都不一样**）

```
祈愿池   → 喂，水！记住我的愿啊！           （对物体喊话，祈使+感叹）
圣油坊   → 油出来了！它差点给油跪下。       （感叹 + 夸张叙述）
圣骸堂   → 它想打喷嚏。憋住了。它真伟大。   （三短句小剧场 + 自夸）
净罪池   → 把昨天扔进水里。咕咚。轻松了！   （三短句 + 拟声 + 感叹）
圣火祭坛 → 那我自己烧会儿。                 （单句自言自语）
禁典阁   → 这书不能看！……它看了。         （自己打自己脸，感叹+省略+反转）
净念殿   → 摘掉一片心事。咦，我来干嘛？     （动作 + 失忆问句）
远客居   → 没客人。它还是烧了水。万一呢？   （短句 + 转折 + 反问）
```

**关键**：8 条 8 种句式，**没有一条是「物体：说话」的冒号对话**。

## 这轮的硬规则

1. **禁用「物体：说话」冒号对话句式**（如 `窑火：？` / `油：哎哟轻点` / `书：你完了`）——上一轮被它毁了
2. **每条句式都要不一样**，可用的电波手法举例：
   - 自己打自己脸（"不能看！……它看了。"）
   - 失忆 / 犯傻（"咦，我来干嘛？"）
   - 三短句小剧场（"想打喷嚏。憋住了。它真伟大。"）
   - 拟声词（"咕咚。" "叮叮叮——"）
   - 自言自语（"那我自己烧会儿。"）
   - 夸张叙述（"它差点给油跪下。"）
   - 转折 + 反问（"它还是烧了水。万一呢？"）
   - 祈使叠句（"抬头！再抬头！"）
   - 意图 vs 结果反差（"想拿酱，翻出了去年。"）
   - 动作被打断
3. **标点放开**：！ ？ …… —— 都用，但别堆
4. 长度 **7-13 个汉字**，绝不超过 13
5. 用"它"指狐狸；用世界观词（狐狸/谷/炉/钢/经/灵脉），不许现实网络词
6. 不要复述建筑描述
