# 任务：为《狐狸谷》典制系统重命名 + 数值对称性

你是资深独立游戏文案，擅长**带画面感、有隐喻、避开学究气**的命名。前一轮 LLM 给的命名（"老狐议事 / 众声堂 / 商道会 / 远风营 / 深谷静修"）**太平、太学究、太"翻译腔"**——本轮要写得**有趣、像狐狸文化、有画面感**。

3 个任务：
1. **6 政体名字**重新润色
2. **POLICY 16 个 opts 名字**重新润色
3. **6 政体效果对称性 + 数值平衡建议**

---

## 一、世界观锁定（命名硬约束）

- 主体：一群**狐狸**在山谷里建文明（不是人类、不是国家）
- 命名要带**狐狸/山谷物件**画面：尾巴、爪、牙、瞳、鬃、骨、霜、雾、橡、岩、苔、月、风、沙、谷、岭
- **避免**：现代政治词（"主义/集议/政体"）、学究气抽象（"传统/制度/秩序"）、翻译腔（"通货集/合嗓令"）
- 字数：**2-4 字**，短而有力
- 风格参考：
  - ✅ **好**："老橡议事"（物件+动作）/ "霜爪令"（自然+器物）/ "嚼根派"（动词+物件）/ "踩月者"（拟物化）/ "拂尘禁"（动作+物件） 
  - ❌ **不要**：守尾制 / 合嗓令 / 通货集 / 老狐议事 / 众声堂 / 商道会（太"翻译腔"）

参考世界文学：
- 极乐迪斯科 Disco Elysium：摩拉莱斯特派（Moralintern）/ 共产主义之蝶
- 密教模拟器 Cultist Simulator：The Long / The Forge / 苍蝇主子（Lord of the Marsh）
- 博尔赫斯：吉诃德派 / 镜中国 / 巴别图书馆

中文文学好例：
- "九节杖" / "霜骨令" / "野渡集" / "苦舟会"

## 五、输出格式

### 任务 1：6 政体 + Tier 1 路线名

每个 ID 给 **3 个候选** + 简短理由（让玩家挑）。格式：

```markdown
| ID | 旧名 | 候选 A | 候选 B | 候选 C | 推荐 | 理由 |
|---|---|---|---|---|---|---|
| elder | 守尾制 | 老橡议 | 鬃议会 | 数尾派 | A | 物件具象 |
| ...
```

每个候选**字数 2-4 字**，**有狐狸/山谷画面感**。3 个候选要明显**不同方向**（不是同义改写）。

### 任务 2：16 政策 opts

按域分组，同样 3 候选 + 推荐 + 理由。

### 任务 3：数值评估 + 调整

简短评估（< 200 字）+ 直接抄代码的调整列表（5-10 条）。
