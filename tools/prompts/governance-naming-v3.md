# 任务：为《狐狸谷》典制系统重命名（中国古典村社志风格）

你是资深独立游戏文案。前两轮 LLM 给的命名要么**翻译腔**（守尾制/合嗓令），要么**兽性过头**（磨牙堂/咬尾犁），要么**信息量不够**（嚼根/亮爪——玩家看不出政体属性）。

本轮要写**中国古典村社志风格** + **看名能猜机制** 的命名。

3 个任务：
1. **6 政体名 + Tier 1 路线名**
2. **POLICY 16 个 opts 名**
3. **6 政体效果对称性 + 数值平衡建议**

---

## 一、风格硬约束（重要！）

### ✅ 要的方向：中国古典村社志

像《村庄志》《山谷志》《十里乡略》那种**朴素、带山水/物件画面、有民间味道**的命名。

参考好例：
- 物件 + 功能词："老橡守"（老橡树+守护）/ "衔铜道"（叼铜钱+商路）/ "喧藤议"（藤蔓喧响+议事）
- 自然 + 动作："破霜行"（踏破寒霜+前行）/ "封霜门"（霜冻封印+大门）/ "聚土穴"（聚集泥土+巢穴）
- 拟物 + 礼仪："露颈礼"（露出颈部=狐狸友好动作+礼节）/ "按须坐"（按胡须长短+落座）

### ❌ 不要的方向

- **翻译腔/学究气**："集议传统 / 通货集 / 合嗓令 / 守尾制"
- **兽性过头**："磨牙堂 / 咬尾犁 / 嚼骨课 / 啃金窟"——玩家觉得不够端庄
- **极简到读不懂**："嚼根 / 吞月 / 铜牙 / 亮爪"（2 字看不出政体属性）
- **西洋风**："摩拉莱斯特派 / 苍蝇主子"
- **后缀套路**：避免 4 个以上候选都用同一后缀（守/修/道/行/堂/会 等）

### 核心约束：**信息密度**

玩家**第一次看到这个名字**，应该能**大致猜出政体核心**：

| 政体 | 玩家应该一眼看出的 | 命名应包含暗示 |
|---|---|---|
| elder（长老学识） | "学者派 / 老人议事" | 老/橡/卷/议/谋/书/朴 |
| hermit（隐修灵性） | "修行派 / 闭关" | 隐/雾/影/灵/月/谷/独/苔 |
| public（公议均衡） | "众议派 / 民主" | 众/议/共/和/聚/合/同/邻 |
| trade（商业铜钱） | "商业派" | 商/贸/货/铜/市/驮/衔/路 |
| martial（远征武力） | "远征派 / 探索" | 远/征/拓/行/野/猎/旅/烽 |
| anarchy（散漫手动） | "无政府 / 散漫" | 散/野/独/无/由/自/纵/任 |

字数：**3 字最佳**（2 字信息不够，4 字啰嗦）。极少数情况可 2 字（如"喧藤"足够具体）或 4 字（如"鬃霜烽火"）。
