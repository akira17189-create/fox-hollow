# 任务：为《狐狸谷》研究重命名（第四批：神启线 39 个）— 神圣肃穆 / 秘仪神秘

《狐狸谷》的研究分几条线，已重命名三批：
- **工业线**：工业科技风（煤铁炼钢、蒸汽动力、辐射屏蔽）
- **灵修线**：中式密教·简洁（触灵、聆灵、磨镜、幽典、渊脉）
- **本批：神启线** —— 宗教 / 教团 / 圣战 / 秘仪

## 方法（沿用灵修线，已验证）

1. **简洁，不堆砌辞藻，不拘字数**（2/3/4 字都行，看哪个最自然贴切）
2. **去掉"教科书后缀"**——神启线现名一堆 "X术 / X学 / X论 / X法"（圣火术、教令学、圣工论、审判论、朝圣论、圣骸学、化神论、飞升论、净念论、神墨术、预言术、灵界论、卜兆论……），这些后缀要去掉或换更自然的说法
3. **不用复杂生僻字**，但可以庄重、有分量
4. **现名已经够好的 → 直接保留**（该行写现名即可）。神启线不少现名是好的——祭祀礼法、经典研习、恩典感召、圣铁锻造、圣典编纂、圣战备战、远征神学、圣地夺还、秘仪入门、禁典研读、化神仪典、禁典编纂、神墨精炼、墨经、远见、触虚、形魄羽化……这些可保留

## 气质

神启线有两面，命名要各自贴合：
- **教团**（初悟 + 教团阶段）：组织化的宗教——圣火、教令、圣战、教会、朝圣。气质**庄严、神圣、肃穆**，有"信仰的重量"。
- **秘仪**（秘仪阶段）：隐秘的神秘修行——化神、飞升、禁典、预言、触虚、灵界。气质**幽秘、玄奥**，接近灵修线的密教感。

## 重名禁区

不能和已定的研究名重复。特别注意：
- `pureMindLore`「净念论」**不能改成「净念」**——灵修线已有研究叫「净念」。换别的（如「除妄」「摘念」之类）。
- 其他也避开已用过的：触灵/寻脉/形魄/幽典/灵契/磨镜/元念……（灵修线已用）
