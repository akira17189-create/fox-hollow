# 任务：为《狐狸谷》12 个灵术各写一句 tip（电波 / 无厘头风格）

《狐狸谷》是中式叙事增量游戏，主角是山谷里的小狐狸。
**灵术**是它施放的法术——瞬间释放、效果或持续一季、或永久。

## 写作风格（延续前几批最终定下的）

- 长度 **7-13 个汉字**，绝不超过 13
- **电波 / 无厘头 / 跳脱 / 有包袱**，标点放开（！？……——）
- **多用无主语 / 省略主语句式**，不要每条都"它怎么怎么"
- **每条句式都不一样**
- **禁用「物体：说话」冒号对话句式**
- 侧写角度：**施法的瞬间 / 施法者的状态 / 法术效果的具象呈现**，不要复述效果数值
- **禁词**：突然 / 忽然 / 能量 / 力量 / 神奇 / 系统 / 解锁 / 激活 / 配方 / 产出 / 效率
- **禁套式**：`不是X，是Y` / `X，是X`
- 用世界观词（狐狸/谷/炉/灵脉/星图/寂石），不许现实网络流行语

## 用户已通过的范例（全部通过审核，钉死风格）

```
喂，水！记住我的愿啊！           （对物体喊话）
把昨天扔进水里。咕咚。轻松了！   （拟声 + 三短句）
那我自己烧会儿。                 （自言自语）
这书不能看！……它看了。         （自己打自己脸）
摘掉一片心事。咦，我来干嘛？     （失忆 / 犯傻）
想偷听秘密，先学会闭嘴。         （格言式，无主语）
击掌。                           （极简，无主语）
审镜子！镜子不说话。             （无厘头）
嗡——山谷抖了三下。             （拟声 + 夸张）
登顶，然后遗忘来路。             （无主语叙述）
它想打喷嚏。憋住了。它真伟大。   （小剧场 + 自夸）
上去的是它。下来的……也说自己是它。（对仗 + 不确定）
翻到底层。摸到寂静。             （无主语动作）
火舌卷走了影子。                 （无主语场景）
预言在梁上发芽。                 （无主语，意象）
```
