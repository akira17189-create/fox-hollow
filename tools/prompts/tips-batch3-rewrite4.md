# 任务：为《狐狸谷》9 座建筑各重写一句 tip（电波 / 无厘头，**多用无主语句式**）

电波方向、句式多样都对了。新问题：**太多句子都以「它」开头叙述**。

## 关键反馈（原话）

> "为什么都要用它来叙述，可以没有主语啊。"

## 用户已通过的 14 条（注意有多少是无主语 / 省略主语的）

```
喂，水！记住我的愿啊！           （无主语，对物体喊）
油出来了！它差点给油跪下。       （前句无主语）
它想打喷嚏。憋住了。它真伟大。   （中句无主语）
把昨天扔进水里。咕咚。轻松了！   （全程无主语！）
那我自己烧会儿。                 （无主语，自语）
这书不能看！……它看了。         （前句无主语）
摘掉一片心事。咦，我来干嘛？     （无主语 + 第一人称）
没客人。它还是烧了水。万一呢？   （前句无主语）
唱完了。梁上还在嗡。它有点小得意。（前两句无主语）
上去的是它。下来的……也说自己是它。
它盖了个章。歪了。它假装那是风格。（中句无主语）
想偷听秘密，先学会闭嘴。         （全程无主语！格言式）

击掌。                           （全程无主语！极简）
```

**重点**："想偷听秘密，先学会闭嘴" 和 "击掌。" 这种**完全没有主语**的，用户特别喜欢。

## 这轮硬规则

1. **优先写无主语 / 省略主语的句子**——9 条里至少一半不要出现"它"
   - 可用：祈使/格言式（"先学会闭嘴。"）、纯动作（"击掌。" "添柴。"）、纯场景（"柴没了，火还烧着。"）、拟声感叹（"咕咚。轻松了！"）、省略主语的叙述（"唱完了。梁上还在嗡。"）
2. **禁用「物体：说话」冒号对话句式**（延续上轮）
3. **每条句式不重复**
4. 长度 **7-13 个汉字**，绝不超过 13
5. 标点放开：！ ？ …… ——
6. 用世界观词（狐狸/谷/炉/钢/经/灵脉），不许现实网络词
7. 不要复述建筑描述
