# 任务：为《狐狸谷》23 座建筑各重写一句 tip（**电波 / 无厘头版**）

前两轮写得太文艺、太"安静"、太拘谨，全被驳回。这轮**彻底放开**。

## 关键反馈（原话）

> "能不能玩点梗，二次元的，无厘头一点的，突然在感叹什么，用感叹号，问号，句号。电波一点的，都可以，你太局限了"

要的就是：**有梗、二次元、无厘头、电波系、跳脱、好玩**。

## 这轮的硬性转变

1. **标点放开**：感叹号！问号？省略号…… 全部可以用，不要只会用句号
2. **可以无厘头**：荒诞、跳跃、自问自答、突然感叹、前后不搭但好笑
3. **可以电波系**：思维跳跃、神经质又可爱、让人噗嗤一下
4. **可以拟人对话**：物体可以"说话"、可以只回一个"？"、可以摆烂
5. **可以有狐狸的内心 OS**：吐槽、纠结、怂、得意、失忆
6. **梗用"世界观内"的方式玩**：不许用现实网络流行语（yyds、绝绝子之类），但**句式节奏、反转、吐槽感**要够网感

## 电波 / 无厘头句式示范（学这种"感觉"，不是抄内容）

```
· 喂，水！记住我的愿啊！                （对物体喊话）
· 三天没添柴了，窑火：？                （物体只回一个标点）
· 它想打喷嚏。憋住了。它真伟大。         （短句小剧场 + 自我夸奖）
· 这书不能看！……它看了。               （自己打自己脸）
· 没椅子？！只好站着把命运听完。         （感叹问号叠用）
· 它睡着了。火：那我自己烧会儿。         （物体接梗）
· 坐了一天，悟了。悟了啥？保密。         （自问自答 + 摆烂）
· 上去一只狐狸，下来一个"？"             （标点本身当包袱）
```

注意：**节奏要碎、要跳、要有"包袱"**。一句话里可以有转折、有反差、有吐槽。

## 仍然要守的

- 长度 **7-13 个汉字**（标点不算字数，但别堆太多），绝不超过 13 字
- 用"它"指狐狸
- 用世界观内的词（狐狸 / 谷 / 炉 / 钢 / 经 / 灵脉……），不许现实网络词
- **禁词**：能量 / 力量 / 神奇 / 系统 / 解锁 / 激活 / 配方 / 产出
- 不要复述下面给的建筑描述
