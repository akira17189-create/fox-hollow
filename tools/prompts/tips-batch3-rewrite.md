## 写作风格

- 长度 **7-13 个汉字**，**绝不超过 13**
- 要：具体动作 / 具体物件 / 意外发现 / 温情 / 巧思 / 反差
- 不要：空灵抽象、纯写气氛、"飘"的句子
- 主语优先：**它（狐狸）做某个具体动作**
- **禁词**：突然 / 忽然 / 能量 / 力量 / 神奇 / 无比 / 沉默下来 / 安静下来
- **禁套式**：`不是X，是Y` / `X，是X`
- **禁机制术语**：解锁 / 激活 / 配方 / 产出 / 效率
- **禁复述 desc**
